
# Fiscaliza Brasil

Sistema para consultar e fiscalizar os Deputados Federais do Brasil usando a API da camara dos deputados

## Funcionalidades

- Consultar dados do deputados
- Consultar leis

## Instalação

Instale com npm (requer node instalado)

```bash
  npm install 
  npm run dev 
```
